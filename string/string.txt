C++中目前只关注string类即可，字符类型的可以不用管，string类可以作为一种专门为字符串设置的数据类型，要使用string类通常要包含string头文件，string类位于命名空间std中
在编程中也可直接用作用域的方式访问
1.使用string类时，可以将一个string对象赋值给另一个string对象，两个string对象也可以用+拼接起来
2.在C++中可以用cin运算符将输入存入string对象中，可以用cout将string对象显示
3.string 类里常用的构造函数
string s1 默认构造函数，空对象
string s2("hello") 有参构造函数，初始化s2 = string s2 = "hello"
string s3(s2)拷贝构造函数
string s4(5,"a")初始化为5个a
string s5(s2,1,3)表示从s2的下标1开始取到下标3
4.常用方法，length(),empty(),clear()
5.访问其元素，类似于数组下标访问
6.查找与替换 find()返回括服里第次出现的下表位置，没有这个元素则返回string::npos 即-1，表示未找到
replace(start,num,"")表示从start开始用“”里替换num个字符
7.子串与插入 substr(start,end)表示获取某个字符串里的子字符窜
insert(index,"")在index位置插入“”
8.遍历string
使用基于范围的for循环
for(auto son : s)
{

    .....
}
使用迭代器
for(auto it = s.begin(): it != it.end(); it++)
{   std::cout<<*it;
    .....
}
9.数值转换
字符串转数字 stoi("")转换为int stod()转换double
数字转字符串 to_string()
string对象中的字符处理，通常使用到cctype头文件里的函数，有
isalnum(c) 当c是字母或者数字时为真
isalpha(c) 是字母时为真
iscntrl(c) 是控制字符时为真
isdigit(c) 是数字时为真
islower(c) 是小写为真
isupper(c) 是大写时为真
tolower(c) 大写转小写
toupper(c) 小写转大写
